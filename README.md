# Hachlef!!
Hachlef!! ( החלף : switch/swap 「ハフラファ(切り替える・交換する)」の命令形 ) is a keyboard language switcher for WebView2 apps.

This is a utility tool to work around the issue where the LAlt+Shift language switching function does not work in WebView2 apps.
It is published for my ["Google Chat Desktop"](https://github.com/KHiyowa/google-chat-desktop), but please feel free to fork and modify it as needed.
It might also be effective for Electron applications.

I have not verified operation with anything other than Google Chat Desktop, so please use it at your own risk.

日本語圏の方へ: 拙作「[Alt-IME 2](https://github.com/KHiyowa/alt-ime-ahk)」とは同時にご利用いただけません。Alt-IME 2はShift + LAltをShift + 無変換に読み替えるため、予期しない動作が発生する可能性があります。
